# Tổng quan dự án

**Tên:** NGS Monitor — Nền tảng Thu thập & Phân tích Nội dung Truyền thông Phòng, Chống Tin Giả tại Việt Nam

**Mục đích:** Web application cho phép nhà nghiên cứu tự động thu thập bài viết từ các báo điện tử và cổng thông tin chính thống của Việt Nam, dùng AI phân loại nội dung vào 8 nhóm chủ đề, phân tích sentiment, trích keyword — rồi tổng hợp thành file báo cáo Word (.docx) theo mẫu nghiên cứu học thuật.

**Phạm vi MVP:**
- Chỉ hỗ trợ website / báo điện tử (20–50 nguồn)
- Mạng xã hội (Facebook, YouTube, TikTok, Zalo) ngoài phạm vi MVP
- Desktop-first UI
- AI chạy local, không gọi API bên ngoài
